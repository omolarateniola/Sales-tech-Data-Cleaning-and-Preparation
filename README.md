# Sales-tech-Data-Cleaning-and-Preparation
This is the dataset of an hypothetical sales company that deals with gadgets.
The dataset contains the following columns;
Order ID, Customer ID, Sales Rep ID, Sales Rep Name, Date, Time, Poduct, Product Category, Unit Price, Unit Cost sales Region, Sales Rep Email, Feedback, Feedback Category, Gros Revenue, Tax Rate, Tax Amount, Discount rate, Discount Amount and Net Sales. 
The data Cleaning was carried out by editing the top row, Filing in blanks, sorting and filtering, using lookup functions and formulas to fill some columns, fixing the date and time..
